# DDnet-CFG

--------------------------------------

📢 Конфиг для игры DDnetworlds

--------------------------------------

📢 Добавляет новые бинды на Numpad

--------------------------------------

📢 KintaFly, HookFly, HammerFly

--------------------------------------

📢 GoresMode, Default.

--------------------------------------

📝 Описания кнопок

--------------------------------------

⌨️ Num0 - Default 
⌨️ Num1 - HammerFly
⌨️ Num2 - HookFly
⌨️ Num3 - KintaFly
⌨️ Num4 - GoresMode
⌨️ Num9 - Command's
⌨️ Key Z - Залипание Дамми
⌨️ Key CapsLock - Градусы
⌨️ Key R - Kill
⌨️ Key PageUp - Показ всех
⌨️ Key PageDown - Показ тайлов

--------------------------------------

📝 Теперь разбор всех Кнопок подробнее

--------------------------------------

📢 Кнопка Num0 убирает раннее включёный мод, вы включили мод HammerFly либо же какой нибудь другой, чтобы отключить его жмём кнопку Num0.

--------------------------------------

📢 Кнопка Num1 после нажатия этой кнопки у вас включаеться мод HammerFly, В этом режиме дамми бьет Хаммером в вашу сторону при нажатию кнопки стрельбы. Так же с этим модом можно делать (deep fly) с замороженным дамми.

--------------------------------------

📢 Кнопка Num2 после нажатия этой кнопки у вас включаеться мод HookFly, эта функция даёт вам удобно летать с дамми Хукфлаем, при нажатие на кнопку стрельбы дамми хукает.

--------------------------------------

📢 Кнопка Num3 после нажатия этой кнопки у вас включаеться мод KintaFly, в этом моде много функций с этим модом можно делать хаммерфлай (база), трипл-флай (3fly) для высокой скорости, псевдофлай (pseudofly) без контакта и глубокий флай. После включения этого мода у вас появляеться еще один бинд на Кнопку W при нажатии на эту кнопку дамми начинает хукать при повторном нажатии перестаёт.

--------------------------------------

📢 Кнопка Num4 после нажатия этой кнопки у вас включаеться мод GoresMode это не очень функцианальный мод он вам особо нечего не даёт, При стрельбе переключаеться на Хаммер, при Хукании переключаеться на Пистолет.

--------------------------------------

📢 Кнопка Num9 после нажатия этой кнопки, у вас в кончсоль прописываються полезные команды для комфортной игры и так же бинды, подробно просмотреть все бренды и команды можно в папке Zook/Command/Com.cfg

--------------------------------------

📢 Кнопка PageUp после нажатия этой кнопки,
у вас включаеться показ всех игроков, тех кто находится в соло режиме, и тех кто находиться в Тиме, так же при повторном нажатии отключаеться показ.

--------------------------------------

📢 Кнопка PageDown после нажатия этой кнопки, у вас показываються тайлы, при повторном нажатии отключаються.

--------------------------------------

📢 Кнопка Z после нажатия этой кнопки, у вас отключаеться залипание кнопок дамми, при повторном нажатии включаеться.

--------------------------------------

📢 Кнопка CapsLock после зажатия этой кнопки, у вас включаються градусы, но чтобы они работали сперва нужно нажать кнопку G один раз после активации конфига
